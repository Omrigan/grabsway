# Grabsway
```
grabsway [-v] [-p] [-u] [-f] [-n] - an util for making screenshots and sharing it on GitHub Pages 
    -p Copy path to clipboard
    -u Copy url to clipboard
    -f Grab focused window
    -n Skip edit
    -v Verbose
```
